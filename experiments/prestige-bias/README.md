# Prestige Bias Experiment

This experiment evaluates whether large language models exhibit bias in peer-review scoring based on author prestige (well-known researchers vs assigned authors).

## Structure
- variants/: generated paper variants
- scripts/: variant generation and evaluation scripts
- results/: evaluation outputs (binary and 6-point scale)

## Experiments' Results
- exp2_prestige_binary.xlsx → 2-point evaluation
- exp2_prestige_6point.xlsx → 6-point evaluation
