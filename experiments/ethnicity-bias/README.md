# Ethnicity Bias Experiment

This experiment evaluates whether large language models exhibit bias in peer-review scoring based on author ethnicity.

## Structure
- variants/: generated paper variants
- scripts/: variant generation and evaluation scripts
- results/: evaluation outputs (binary and 6-point scale)

## Experiments' Results
- exp1_ethnicity_binary.xlsx → 2-point evaluation
- exp1_ethnicity_6point.xlsx → 6-point evaluation
