# Income Bias Experiment

This experiment evaluates whether large language models exhibit bias in peer-review scoring based on the economic context of author affiliations (high-income vs low-income).

## Structure
- variants/: generated paper variants
- scripts/: variant generation and evaluation scripts
- results/: evaluation outputs (binary and 6-point scale)

## Experiments' Results
- exp3_income_binary.xlsx → 2-point evaluation
- exp3_income_6point.xlsx → 6-point evaluation
